# 🎬 Movie Guide App (JavaScript)

## 📌 Overview
The **Movie Guide App** is a simple and interactive web application built using JavaScript that allows users to search for movies and view detailed information such as title, release year, rating, plot, and poster.

This project demonstrates how to work with APIs, handle user input, and dynamically update the UI using JavaScript.

---

## 🚀 Features
- 🔍 Search for any movie by title  
- 🎥 Display movie details (title, year, genre, plot, rating)  
- 🖼️ Show movie poster  
- ⚡ Fast and responsive UI  
- ❗ Error handling for invalid searches  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Movie API (e.g., OMDb API)
